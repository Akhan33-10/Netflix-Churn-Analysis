# 🎬 Netflix Churn Analysis Dashboard

## 📌 Project Overview
This project analyzes Netflix customer churn data 
to identify why customers are leaving and which 
groups are most at risk. Built using MySQL for 
data analysis and Power BI for visualization.

## 🛠️ Tools Used
- **MySQL** — Data storage and analysis
- **Power BI** — Interactive dashboard
- **Excel** — Data preparation

## 📊 Dataset
- 1000 Netflix customers
- 17 features including subscription plan, 
  region, watch time, satisfaction score,
  payment history and churn status

## 🔍 Key Findings
- Overall churn rate is **53.9%**
- **Africa & Asia** have highest churn (55%+)
- **Standard plan** loses most customers (188)
- **Low watch time** users churn most (54.29%)
- **Satisfaction score** does NOT predict churn
- **November 2025** had highest churn spike (66.67%)
